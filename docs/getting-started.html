Getting Started

Takes about 10 minutes the first time, mostly waiting for downloads. After that, using it takes seconds. No technical knowledge, GitHub account, or IT department required — just a Windows computer and an internet connection.

1Download the installer

Go back to the home page and click the big "Download for Windows" button. Your browser will save a file — usually to a folder called Downloads. Most browsers show it at the bottom of the window or in a small pop-up when it finishes; you can click that to open it directly, or open your Downloads folder and look for a file named something like DentiScribe.AI.Setup.0.1.0.exe.

2Run it

Double-click that downloaded file (click twice quickly with the left mouse button) to start it.

You'll see a blue "Windows protected your PC" screen. This is completely normal for free software from an independent developer — it's not a virus warning, it just means the file hasn't gone through Microsoft's paid certification program (which costs money and isn't worth it for a free app).

Click the small "More info" text (easy to miss, it's not a button), then click the "Run anyway" button that appears below it.

The app installs itself automatically — there's nothing to choose, no folders to pick, no options to configure. A window titled "DentiScribe AI" will open by itself when it's done, usually within a few seconds.

3Create your account

First time you open it, you'll be asked to create an account — just pick any username and password for this app. It's stored only on this computer and isn't shared with anyone, including us. Type a username, type a password, click "Create Account." Remember these — there's no "forgot password" link, since nothing is stored anywhere except your own computer.

4Run the one-time setup

A setup screen appears automatically the first time. It installs two small, free pieces of software the app needs to run its AI locally:

  • Ollama — the note-formatting engine
  • Whisper — the transcription engine

Click each "Install" / "Download" button in order and wait for the progress bar to finish before clicking the next one. This downloads a few gigabytes total, so it's worth doing on a decent internet connection — plug into Wi-Fi and maybe grab a coffee. You only need to do this once, ever.

If something fails partway through, click "Skip for now" — you can reopen this screen anytime from the Setup button at the top of the app and pick up where you left off.

"Install Ollama" or "Install Whisper Engine" seems stuck or failed? Everything here is 100% free and runs on your own computer — there's no paid alternative, and you shouldn't need one. Try these in order:

  1. Just click the button again. Most failures are a slow or temporarily-interrupted download — clicking "Install Ollama" or "Install Whisper Engine" a second time picks up where it left off or retries cleanly.
  2. Check your internet connection is stable — these downloads are a few gigabytes total, so a Wi-Fi connection that drops out partway through can interrupt them. A wired connection or moving closer to your router helps.
  3. Check you have enough free disk space — right-click your C: drive in File Explorer → Properties, and make sure you have at least 10GB free. Delete some files (or empty the Recycle Bin) if not.
  4. Restart the app — close DentiScribe AI completely and reopen it, then click Setup at the top to bring the install screen back and try again.
  5. Still stuck? Email McMusclesO@gmail.com with a description of what happened (which button, what message appeared) and we'll help directly — no charge, ever.

What Ollama and Whisper actually are, in plain terms: they're two small background programs that do the AI work — Ollama turns your transcript into a formatted note, Whisper turns recorded audio into text. You never open or interact with them directly; the DentiScribe AI window is the only thing you use. Once installed, they quietly run in the background whenever DentiScribe AI needs them, using no internet and sending nothing anywhere.

5Teach it your style

In the main screen, find "My Style Examples." Paste in 2-3 of your own previously-written notes (formatted however you normally write them). The AI uses these as a reference so its output matches how you actually write, not a generic template. You can skip this and add examples later — the app works without it, just with more generic formatting.

6Use it

  1. Click "Start Recording" during or after an encounter (or just paste in a transcript if you already have one)
  2. Click "Stop Recording" when done — it transcribes automatically
  3. Click "Format Note"
  4. Click "Copy" and paste the result into EXACT (or wherever it needs to go)

Always read the formatted note before using it. This is a drafting aid, not a replacement for your own clinical judgment.

Troubleshooting

The setup steps keep failing. Usually means antivirus or a restrictive network is blocking downloads. Try again on a different network, or ask whoever manages your practice's computers to allow this app through the firewall.

Recording says "Could not access the microphone." Windows may be blocking mic access. Check Windows Settings → Privacy & Security → Microphone, and make sure "Let apps access your microphone" is on.

Antivirus flagged the installer. This can happen with any new, unsigned app (ironically because malware also often uses the same underlying framework this app is built on). If you're unsure, ask your IT person to verify the download source before running it.
